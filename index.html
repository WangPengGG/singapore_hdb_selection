1一句话结论

买家手头 ~S$0.9M 现金(首付+税+装修)、月入 >S$18k，三套均买得起，差别在取舍：

  • 要学区 + 长地契 + 最大实用面积 → Clementi Crest（~S$1.50M）：综合最契合 40岁2孩首套自住，首推。

  • 要 CBD 核心地段 + 地标景观、且不看重学区 → Pinnacle@Duxton（~S$1.56–1.68M）：情怀地段之选；但短地契 + 弱学区 + 高价高点三重不利,且 4% 压测下 MSR 可能吃紧。

5.3 Pinnacle@Duxton（D2）：持续破纪录的"奖杯"项目

  • 4房记录阶梯(~3 年 +24%)：S$1.228M(2022/4 首破百万) → S$1.41M / 1,409 psf(2023/11，全国首宗转售破 1,400 psf) → S$1.518M(2025/3)。
  • 5房记录阶梯(~2 年 +16%)：S$1.48M(2023) → S$1.58M / 1,398 psf(2025/6) → 历史最高 S$1.63M / 1,442 psf(2026/5，Blk 1B 43–45 楼)。
  • 均价 PSF 从 ~S$1,203(2023)升至 ~S$1,321(2025)；自 2014 MOP 以来均价 +44%(跑赢全国 +33%)；全项目仅 ~1,848 套，稀缺。
  • 展望：仍是高端标杆并偶有破纪录，但增速放缓、更具选择性。高层 5房或测 ~S$1.7M，但均价仅低个位数年增；近天花板、上行更有限且依赖偶发新纪录。

5.5 对你这次决策的启示

  • 时机：大盘从 2024 +9.7% 降到 2025 ~+3%、2026 Q1 −0.1%，叠加 MOP 供应近翻倍——买方窗口，可从容议价，不必追涨抢购。
  • 保值：重长持 / 资本保全 / 未来最易转手 → Crest 的 ~94 年租约更优；重中央地标 / 已验证奖杯资产 → Pinnacle 地段与头条价更强(但入场更贵、租约短 ~10 年、升值空间更薄)。
  • 地契风险：三盘此持有期内均不临近 60 年融资悬崖，租约差异主要影响 20 年以上的终值。
  • 数据注意：单季高波动(如 Clementi 2025 Q1 +15.4% 环比)与受结构影响的全镇中位数(2024 Q4 S$745K)不宜当趋势；利率与潜在新冷却措施是变量。

6财务方案（PR 首套 · 现金 ~S$0.9M · 月入 >S$18k · 多首付 + 短贷款）

PR 规则(2026现行,已核实):ABSD 首套5%;无任何 HDB/CPF 补贴;只能银行贷款(LTV≤75%、最低5%现金首付);买 HDB 须名下无其他房产;MSR 30% + TDSR 55%,银行按 ~4% 压力利率核额。来源 IRAS / MAS / HDB。

你的方向：多首付 + 更少更短的贷款。 这样做有三重好处——① 省利息(短年限利息可省一半)、② 约 55 岁前还清(40岁 + 15年)、③ 降月供更易过 MSR。三个硬约束:
• 首付至少 25%(LTV≤75%)、其中≥5% 须现金;　• 40 岁想保住 75% LTV 资格,贷款年限宜 ≤25 年(你本就要更短,不冲突);　• MSR 30% 是缩短年限的硬天花板——年限越短月供越高,需月入越高。

7推荐与决策理由

🥇 综合首选：Clementi Crest（~S$1.50M）—— 学区 + 长地契优先

理由：地契最新(~94年,长期持有/再融资几乎无地契衰减顾虑);学区三套中最强(隔壁 Clementi Primary + 南华 SAP 名校,对2个孩子价值极高);113㎡ / 1,216 sqft 是三套中实用面积最大的5房;2032 年跨岛线(CRL)换乘升级是确定性利好。

代价：价格已快速上行至记录区(10个月内 S$1.458M→1.58M),议价空间有限;但买后现金缓冲 ~S$31万仍充裕。把"孩子教育 + 长期自住保值"放第一的家庭,Crest 最契合。

🥈 价值 / 稳健之选：Trivelis（~S$1.28M）—— 最大缓冲 + 最低风险

理由：三套中最便宜,买后留 ~S$40万现金缓冲(可覆盖装修、2孩教育、应急或加速还贷);同样 7 分钟到 Clementi MRT、成熟市镇、地契~87年对40岁夫妻终身无忧;明显便宜于隔壁 113㎡ 的新组屋(已成交 1.5–1.58M);还有 ~2–3% 议价空间。

代价：实用内部面积偏小(~88㎡ / 947 sqft)对4口偏紧、走廊较窄;DBSS 有历史质量缺陷记录(看房须逐项验收);5房成交稀薄。在意财务安全垫与抗风险胜过学区者,Trivelis 性价比最高、下行风险最小。

🥉 情怀 / 地段之选：Pinnacle@Duxton（~S$1.56–1.68M）—— 仅当地段是核心诉求

理由：地段无可替代(CBD、Outram 三线换乘、无遮挡海湾/CBD 天际线、50 楼空中连廊);蓝筹保值、流动性强、连续刷新记录。

代价：总价最高 + 接近历史高点;地契最短(~84年,已用掉~15年)对要持有数十年的家庭是真实长期拖累;学区最弱(1km 内基本仅 Cantonment Primary),对2个孩子是明显短板;现金缓冲最薄(~S$28万)且 4% 压测下 MSR 可能吃紧。除非无强学区需求、就要市中心地标自住,否则对2孩家庭排第三。

8议价空间（基于实际成交，非乐观叫价）

  • Trivelis ~S$1.28M：高于近期成交(2026/6 中层 S$1.243M、2025/8 S$1.275M),约有 2–3% 议价空间,可压向 ~S$1.24–1.26M;只有真正高层/无遮挡景观才值 ~S$1.28–1.30M。
  • Crest ~S$1.50M：基本处于近期成交地板。中低层(25–27楼)2025/8 成交 S$1.458M → 对中低层 ~1.50M 偏进取,可议至 ~S$1.45–1.46M;但对高层(37楼+),对照 5 月 S$1.58M 记录反而偏吸引、议价小甚至可能抢 offer。务必锚定"楼层匹配 + 近3–6个月"成交。
  • Pinnacle ~S$1.56–1.68M：议价空间窄。5房稀缺、需求记录级,高层/翻新/无遮挡盘常近叫价成交;仅低层/未翻新盘才有较大空间。叫价高于 HDB 估值时,COV 差额需全现金支付。

行动建议：三套都先做贷款预批(IPA)确认 MSR/TDSR(尤其 Pinnacle);出价前在 HDB resale portal 核对该单位精确地契起始日,及该栋的 SPR 配额(每栋约8% / 每邻里约5%)与 EIP(PR 买家硬约束);ABSD/BSD 按"价格与 HDB 估值孰高"计。

9注意事项 / 红旗

  • 🚩 SPR 配额 + EIP:PR 买家受"每栋 SPR 配额"与族群配额双重限制——出价前在 HDB 确认该单位对你们开放(Crest 445A 列明"开放所有种族")。
  • 🚩 地契:Pinnacle ~84年(已用掉~15年),对要长期持有的家庭是真实长期拖累;Crest ~94年最优。
  • 🚩 COV(现金溢价):ABSD/BSD 按"价格与 HDB 估值孰高"计;叫价高于估值的差额需全现金支付。Pinnacle 高层/Crest 高层尤其要注意。
  • 🚩 议价锚点:务必以"楼层匹配 + 近3–6个月同栋成交"议价,别被乐观叫价带偏(见对比表成交印记)。
  • 🎓 学区(PR):PR 子女小一报名优先级低于公民,1km 溢价对你们价值有限;学龄前孩子可在 PR 阶段碰运气,别当硬指标。
  • 🏗️ Trivelis 验房:DBSS 有历史质量缺陷记录,看房逐项验收具体单位状况。
