1一句话结论

买家手头 ~S$0.9M 现金(首付+税+装修)、月入 >S$18k，三套均买得起，差别在取舍：

  • 要学区 + 长地契 + 最大实用面积 → Clementi Crest（~S$1.50M）：综合最契合 40岁2孩首套自住，首推。
  • 要最低风险 + 最大现金缓冲 + 性价比 → Trivelis（~S$1.28M）：稳健价值之选，强力次选。
  • 要 CBD 核心地段 + 地标景观、且不看重学区 → Pinnacle@Duxton（~S$1.56–1.68M）：情怀地段之选；但短地契 + 弱学区 + 高价高点三重不利,且 4% 压测下 MSR 可能吃紧。

6注意事项 / 红旗

  • 🚩 SPR 配额 + EIP:PR 买家受"每栋 SPR 配额"与族群配额双重限制——出价前在 HDB 确认该单位对你们开放(Crest 445A 列明"开放所有种族")。
  • 🚩 地契:Pinnacle ~84年(已用掉~15年),对要长期持有的家庭是真实长期拖累;Crest ~94年最优。
  • 🚩 COV(现金溢价):ABSD/BSD 按"价格与 HDB 估值孰高"计;叫价高于估值的差额需全现金支付。Pinnacle 高层/Crest 高层尤其要注意。
  • 🚩 议价锚点:务必以"楼层匹配 + 近3–6个月同栋成交"议价,别被乐观叫价带偏(见对比表成交印记)。
  • 🎓 学区(PR):PR 子女小一报名优先级低于公民,1km 溢价对你们价值有限;学龄前孩子可在 PR 阶段碰运气,别当硬指标。
  • 🏗️ Trivelis 验房:DBSS 有历史质量缺陷记录,看房逐项验收具体单位状况。
